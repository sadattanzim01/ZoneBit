ZoneBit

Find your timezone, bit by bit.

Uses 24-bit bitmasks — one bit per GMT offset — to match cities to timezones via bitwise AND. No === comparisons.

    💡 How It Works

    Each city is assigned a 24-bit number with exactly one bit ON:

    GMT offset → bit position = offset + 12
        Moscow  (GMT+3) → bit 15 → 000000000100000000000000
        Paris   (GMT+2) → bit 14 → 000000000010000000000000
        New York(GMT-4) → bit  8 → 000000000000000100000000

    To search, a mask is built for the target timezone. A bitwise AND & is applied:

    city.bits & searchMask !== 0  →  MATCH ✓
                        city.bits & searchMask === 0  →  NO MATCH ✗
